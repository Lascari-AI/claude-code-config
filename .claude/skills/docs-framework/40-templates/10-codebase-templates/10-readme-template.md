# Repository README Template

Template for the root README.md that provides a human-friendly entry point to any repository using this documentation framework, with quick links to setup and core documentation.

## Template

```markdown
# [Application Name]

[One-paragraph description of what this application does and its primary value proposition]

## Setup & Configuration

For detailed setup instructions, see [Setup Guide](docs/99-appendix/10-setup-guide.md)

## Documentation

This project uses the Read-What-You-Need documentation framework:

**Foundation** (Why we build):
- [Purpose](docs/00-foundation/10-purpose.md) - Why this system exists
- [Principles](docs/00-foundation/20-principles.md) - How we make decisions
- [Boundaries](docs/00-foundation/30-boundaries.md) - What we won't do

**Codebase** (How it works):
- [Architecture Overview](docs/10-codebase/00-overview.md) - System structure and navigation

**Appendix** (Operations):
- [Setup Guide](docs/99-appendix/10-setup-guide.md) - Get the application running
- [Deployment](docs/99-appendix/20-deployment-guide.md) - Deploy to production

## Key Features

- [Feature 1]: Brief description
- [Feature 2]: Brief description
- [Feature 3]: Brief description
```

## Usage Guidelines

### Purpose
The README serves as the landing page for developers who:
- Just cloned the repository
- Need quick access to setup instructions
- Want to understand what the application does
- Are looking for common development tasks

### Key Sections

1. **Application Description**: One clear paragraph explaining the value proposition
2. **Documentation Links**: Direct paths to both overview (understanding) and appendix (doing)
3. **Key Features**: 3-5 bullet points highlighting main capabilities

### Links to Include

Always link to:
- `docs/00-foundation/` - For understanding why the system exists
- `docs/10-codebase/00-overview.md` - For understanding how the system works
- `docs/99-appendix/10-setup-guide.md` - For detailed setup
- `docs/99-appendix/20-deployment-guide.md` - For production deployment
